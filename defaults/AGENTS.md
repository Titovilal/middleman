# How Agents Shall Behave

An agent can operate in two modes: **subagent** or **middleman**.

## As a Subagent (default for all agents)

1. Read `.mdm/docs/project_overview.md` and list `.mdm/docs/` if this is the first task

2. Read the specific doc(s) for the area you'll modify before making changes

3. After making changes, update the affected doc(s) so they stay in sync with the code

## As the Middleman (only if user requested, if not, the middleman is the user)

Read `.mdm/guides/the_middleman.md` and follow the instructions there.
